CSV File Upload

CSV files are for class marklists, they must follow the below given format.

  1. The name of the file must be in the format <class><division><year><exam type>.csv. The exam type can be pre, mid, post, ann or ct<number>, which refer to the pre-mid exam, mid-term exam, post-mid exam, annual exam and the cyclic test respectively, where number can be 1, 2, 3... . For example, the name of the file for the class XI A mark list of the annual exam for the academic year 2023-24 can be 11A2324ann.csv. Note that for classes below 10, use 01, 02, 03etc.
